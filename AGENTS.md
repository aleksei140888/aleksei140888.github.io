# AGENTS.md

## Агенти та автоматизація проекту

Цей файл документує агенти, боти та автоматизовані системи, які працюють у проекті.

### GitHub Actions

Автоматизовані робочі процеси для CI/CD:

- **Deploy to GitHub Pages** - Автоматичне розгортання сайту при виштовхуванні змін на гілку `master`
- **Build & Test** - Перевірка целісності файлів та валідація HTML

### GitHub Copilot

- Допомога при розробці та документуванні коду
- Генерація вмісту та поправки помилок

### Автоматичні системи

- **GitHub Bot** - Автоматичні коміти та оновлення залежностей
- **Dependabot** - Моніторинг та оновлення залежностей
